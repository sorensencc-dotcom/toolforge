# Tool Lifecycle Manager

Manages skill lifecycle: registration, distribution, validation, deprecation.

## Installation

Requires: Node 18+, TypeScript

## Usage

```bash
npx toolforge skill-manager --action <register|distribute|validate|deprecate>
```

## Metadata

- **Category**: automation
- **Owner**: soren
- **Runtime**: TypeScript
- **Timeout**: 30s
