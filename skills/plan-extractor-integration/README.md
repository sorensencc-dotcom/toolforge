# Plan Extractor Integration

Unified entry point for CodeFlow extraction into CIC stores.

## Quick Start

```bash
npm test
```

## What it does

- Extracts node, edge, pattern, security, and impact metadata from CodeFlow analyzers
- Writes extracted facts into the central CIC store
- Maintains schema alignment with Toolforge governance
