# analyze-token-burn

Project month-end token costs and identify high-spend adapters.

## Quick Start

```bash
analyze-token-burn [mode]
```

- `daily` — Today's burn rate (default)
- `monthly` — Project month-end spend
- `by-adapter` — Top 5 cost drivers
