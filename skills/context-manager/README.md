# Context Manager

Detect and manage autonomous execution context.

## Quick Start

```bash
npm test
```

## What it does

- Detects autonomous execution via `AUTONOMOUS_EXECUTION` env var
- Validates and persists context to `.autonomous-context.json`
- Prevents prompts during long-running tasks
