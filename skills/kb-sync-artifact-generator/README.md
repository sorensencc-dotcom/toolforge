# KB Sync Artifact Generator

Generate interactive knowledge base sync reports with sortable tables, charts, and searchable results.

## Quick Start

```
Generate an interactive KB sync report with sortable broken links table.
```

## What it does

- Runs full KB sync pipeline (wiki scan + integration layer)
- Parses broken links and generates impact-scored findings
- Creates self-contained HTML artifact with sortable Grid.js table and Chart.js visualization
- Updates persisted Cowork artifact for historical tracking
