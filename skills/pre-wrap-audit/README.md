# pre-wrap-audit

12-point blind-spot assessment before session termination.

## Quick Start

```bash
/pre-wrap-audit Phase 27 Wave F deployment
```

## What it does

- Runs structured interview on gaps, risks, assumptions, verifications
- Assesses 4 core + 8 extended fields for deployment readiness
- Returns RED/YELLOW/GREEN verdict for deployment gates
- Auto-triggered on `/finish`; optional on timeout
