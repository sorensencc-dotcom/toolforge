# Ashfall

Deterministic session-termination and context-handoff engine for Cast Iron systems.

## Quick Start

```bash
ashfall --scope=full
```

## What it does

- Gather modified files, architectural deltas, uncommitted changes
- Incinerate irrelevant context, compress memory, prune drift
- Apply Four Questions audit to surface blind spots
- Generate deterministic memory manifest with atomic write integrity
- Rank roadmap and emit pruned token-bounded memory for next session
