# CIC Section Summarizer

Auto-summarize CIC phase progress and file coverage.

## Quick Start

```bash
npm test
```

## What it does

- Analyzes file paths to estimate project completion
- Identifies coverage gaps (files missing unit tests)
- Generates phase progress summaries
