# Rewrite Labs Orchestrator

Orchestrates multi-stage development pipelines and handles blockers.

## Quick Start

```bash
npm run orchestrate -- pipeline.json
```

## What it does

- Calculates percent completion across pipeline stages
- Identifies blocked stages and root causes
- Suggests next execution steps based on state
- Returns actionable recommendations for unblocking
