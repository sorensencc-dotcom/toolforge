# Permission Governor

Auto-approve whitelisted operations and analyze approval bottlenecks.

## Quick Start

```bash
npm test
```

## What it does

- Whitelists safe/trusted operations (read-only queries, shell utilities)
- Auto-approves whitelisted ops to reduce operator prompts
- Analyzes approval cache and bottlenecks
